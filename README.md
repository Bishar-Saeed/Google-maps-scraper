# Complete Email Scraper - Google Maps Business Email Extractor

[![Python Version](https://img.shields.io/badge/python-3.8+-blue.svg)](https://python.org)
[![Playwright](https://img.shields.io/badge/Playwright-1.40+-green.svg)](https://playwright.dev)

## 📋 Description

A powerful, optimized web scraper that extracts business emails from Google Maps listings. This tool:

- **Scrapes Google Maps** for businesses based on profession and location
- **Checks ALL website pages** (unlimited crawling) for email addresses
- **Always checks Facebook** pages for additional contact emails
- **Gets ALL Google Maps results** (up to 120 per search) with improved scrolling
- **Saves data** to Excel files with comprehensive email categories

## ✨ Features

- 🎯 **Smart Google Maps Scraping**: Scrolls through all results automatically
- 🔍 **Complete Website Scanning**: Discovers and checks ALL pages on business websites
- 📘 **Facebook Integration**: Always checks Facebook pages for emails
- 📊 **Excel Export**: Saves organized data with multiple email categories
- ⚡ **Optimized Performance**: Balanced speed to avoid blocks while being efficient
- 🛡️ **Anti-Detection**: Uses realistic browser behavior and delays
- 🔄 **Retry Logic**: Automatically retries failed extractions
